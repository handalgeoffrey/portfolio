# R. Handal Geoffrey - Portfolio

A premium, modern portfolio website built with React, TypeScript, and Tailwind CSS.

## Features
- **Premium Dark Theme**: Deep "Midnight" color palette with vibrant accents.
- **Glassmorphism**: Frosted glass effects on Navbar and cards.
- **Interactive Elements**: Framer Motion animations and magnetic buttons.
- **Dynamic Gallery**: 3D dome gallery view.
- **Responsive Design**: Fully optimized for all devices.

## Tech Stack
- React
- TypeScript
- Tailwind CSS
- Framer Motion
- Vite

## Deployment
Hosted on GitHub Pages via GitHub Actions.
